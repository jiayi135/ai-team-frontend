# 设计风格分析

## ChatGPT (OpenAI)

### 布局
- 极简中心布局
- 大量留白
- 输入框居中突出

### 配色
- 纯白背景 (#FFFFFF)
- 黑色文字 (#000000)
- 绿色品牌色（按钮、图标）

### 字体
- 无衬线字体
- 大字号标题 "What can I help with?"
- 清晰易读

### 交互
- 单一输入框
- "Ask anything" 占位符
- 语音输入按钮
- 附件上传按钮

### 特点
- 极度简洁
- 聚焦对话
- 无干扰元素


## Claude (Anthropic)

### 布局
- 简洁的登录页
- 清晰的产品介绍
- 卡片式功能展示

### 配色
- 橙棕色品牌色 (#CC785C 类似)
- 白色背景
- 温暖友好的色调

### 字体
- 现代无衬线字体
- 大标题 "The AI for problem solvers"
- 清晰的层次结构

### 交互
- "Continue with Google" 主按钮
- 简单的邮箱登录
- 清晰的定价表格

### 特点
- 温暖专业
- 清晰的价值主张
- 友好的用户体验
