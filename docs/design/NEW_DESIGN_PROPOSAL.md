# AI Team Frontend 新设计方案

## 设计理念

本设计方案融合了五大顶级产品的设计精髓，打造一个以对话为核心的现代化 AI 团队协作平台。

### 设计原则

**简洁至上**：借鉴 Apple 和 OpenAI 的极简主义，去除一切不必要的视觉元素，让用户专注于核心功能——与 AI 对话。

**温暖专业**：采用 Claude 的温暖色调和友好交互，同时保持 Google 的专业感，让 AI 协作变得亲切而可信。

**对话优先**：参考 Manus 的对话界面设计，将聊天体验作为整个产品的核心，其他功能围绕对话展开。

**现代科技感**：融入 OpenAI 和 Google 的现代设计语言，使用流畅的动画、渐变效果和微妙的阴影，营造未来感。

## 配色方案

### 主色调

**背景色系**：
- 主背景：`#FFFFFF` (纯白)
- 次级背景：`#F8F9FA` (极浅灰)
- 卡片背景：`#FFFFFF` with shadow

**品牌色系**：
- 主品牌色：`#6366F1` (现代紫蓝，科技感)
- 辅助色：`#EC4899` (活力粉，强调元素)
- 成功色：`#10B981` (绿色，积极反馈)

**文字色系**：
- 主文字：`#111827` (深灰黑)
- 次级文字：`#6B7280` (中灰)
- 占位符：`#9CA3AF` (浅灰)

**对话气泡**：
- 用户消息：`#6366F1` (品牌色)
- AI 消息：`#F3F4F6` (浅灰背景) + `#111827` (深色文字)

### 渐变效果

**Hero 区域渐变**：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**按钮悬停渐变**：
```css
background: linear-gradient(135deg, #6366F1 0%, #8B5CF6 100%);
```

## 布局设计

### 主页布局

**Hero 区域**（全屏）：
- 居中的大标题："AI Team - 你的智能协作伙伴"
- 副标题：简洁的产品描述
- 主输入框：大而显眼，带有渐变边框
- 快速操作按钮：工具生成、技能中心、自我进化

**功能卡片区**（滚动后出现）：
- 三栏网格布局（响应式）
- 每个功能一张卡片，带图标和简短描述
- 悬停效果：轻微上浮 + 阴影加深

**底部区域**：
- 简洁的链接：文档、GitHub、关于
- 版权信息

### 聊天界面布局

**顶部导航栏**（固定）：
- Logo + 项目名称（左侧）
- 功能切换按钮（中间）：聊天、工具、技能、进化
- 用户头像 + 设置（右侧）

**对话区域**（主体）：
- 最大宽度 `800px`，居中显示
- 消息气泡：用户消息右对齐（蓝色），AI 消息左对齐（灰色）
- 时间戳：小字号，浅色
- 代码块：深色主题，带复制按钮
- 流式输出：打字机效果

**输入区域**（底部固定）：
- 多行文本框，自动扩展（最多 5 行）
- 左侧：附件按钮、表情按钮
- 右侧：发送按钮（品牌色，圆形）
- 占位符："和 AI Team 聊天..."

**侧边栏**（可折叠）：
- 历史对话列表
- 快速模板
- 最近使用的工具/技能

## 组件设计

### 按钮

**主按钮**：
- 背景：品牌色渐变
- 圆角：`8px`
- 内边距：`12px 24px`
- 字体：`14px` 加粗
- 悬停：轻微上浮 + 阴影

**次级按钮**：
- 背景：透明
- 边框：`1px solid #E5E7EB`
- 悬停：背景变为 `#F9FAFB`

**图标按钮**：
- 圆形，`40px` 直径
- 悬停：背景色变化

### 输入框

**文本输入**：
- 边框：`1px solid #E5E7EB`
- 圆角：`12px`
- 内边距：`12px 16px`
- 聚焦：边框变为品牌色，带外发光

**多行输入**：
- 最小高度：`56px`
- 最大高度：`200px`
- 自动扩展

### 卡片

**功能卡片**：
- 背景：白色
- 圆角：`16px`
- 阴影：`0 4px 6px rgba(0, 0, 0, 0.07)`
- 悬停：`0 10px 15px rgba(0, 0, 0, 0.1)` + 上移 `4px`
- 内边距：`24px`

**消息卡片**：
- 用户消息：品牌色背景，白色文字，右对齐
- AI 消息：浅灰背景，深色文字，左对齐
- 圆角：`16px`（外侧），`4px`（内侧尖角）
- 最大宽度：`70%`

### 图标

**图标库**：Lucide Icons（已集成）

**使用规范**：
- 大小：`20px` (小)，`24px` (中)，`32px` (大)
- 颜色：继承父元素或使用品牌色
- 一致性：所有图标来自同一图标库

## 动画效果

### 页面加载

**淡入动画**：
- 主标题：从上方淡入，`0.6s`
- 输入框：从下方淡入，`0.8s`，延迟 `0.2s`
- 功能卡片：依次淡入，每个延迟 `0.1s`

### 交互动画

**按钮点击**：
- 轻微缩放 `scale(0.95)`
- 持续时间：`0.1s`

**卡片悬停**：
- 上移 `4px`
- 阴影加深
- 过渡时间：`0.3s`

**消息发送**：
- 消息从底部滑入
- 透明度从 `0` 到 `1`
- 持续时间：`0.4s`

**打字机效果**：
- AI 回复逐字显示
- 光标闪烁

### 过渡效果

**页面切换**：
- 淡出当前页面 `0.2s`
- 淡入新页面 `0.3s`

**侧边栏展开/折叠**：
- 宽度变化 `0.3s`
- 内容淡入/淡出

## 响应式设计

### 断点

- **移动端**：`< 640px`
- **平板**：`640px - 1024px`
- **桌面**：`> 1024px`

### 适配策略

**移动端**：
- 单栏布局
- 侧边栏改为底部抽屉
- 输入框占满宽度
- 功能卡片堆叠显示

**平板**：
- 两栏布局
- 侧边栏可折叠
- 对话区域适当缩小

**桌面**：
- 三栏布局（可选）
- 侧边栏固定显示
- 对话区域最大宽度 `800px`

## 可访问性

### 键盘导航

- 所有交互元素可通过 Tab 键访问
- Enter 键发送消息
- Esc 键关闭模态框

### 屏幕阅读器

- 所有图标带 `aria-label`
- 语义化 HTML 标签
- 合理的标题层级

### 对比度

- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1

## 技术实现

### 使用的库

- **React 18**：UI 框架
- **TypeScript**：类型安全
- **Tailwind CSS**：样式框架
- **Lucide Icons**：图标库
- **Framer Motion**：动画库（可选）

### 关键组件

1. **Hero.tsx**：主页 Hero 区域
2. **ChatInterface.tsx**：聊天界面主组件
3. **MessageBubble.tsx**：消息气泡组件
4. **InputBox.tsx**：输入框组件
5. **Sidebar.tsx**：侧边栏组件
6. **FeatureCard.tsx**：功能卡片组件
7. **Button.tsx**：按钮组件（多种变体）

### 状态管理

- **React Context**：全局状态（用户信息、主题）
- **useState/useReducer**：组件状态
- **自定义 Hooks**：复用逻辑（useChat, useWebSocket）
