## 项目概述

本次设计重构的核心目标是打造一个融合顶级产品设计风格的现代化 AI 团队协作平台，以对话为核心，提供简洁优雅的用户体验。

## 设计理念

### 设计原则

本次重构遵循以下四大设计原则：

**简洁至上**：借鉴 Apple 和 OpenAI 的极简主义，去除一切不必要的视觉元素，让用户专注于核心功能——与 AI 对话。大量留白和清晰的视觉层次让界面呼吸感十足。

**温暖专业**：采用 Claude 的温暖色调和友好交互，同时保持 Google 的专业感，让 AI 协作变得亲切而可信。紫蓝色渐变既有科技感，又不失亲和力。

**对话优先**：参考 Manus 的对话界面设计，将聊天体验作为整个产品的核心，其他功能围绕对话展开。用户可以通过自然语言与 AI 团队交互，实现任务闭环。

**现代科技感**：融入 OpenAI 和 Google 的现代设计语言，使用流畅的动画、渐变效果和微妙的阴影，营造未来感。每个交互都经过精心设计，确保流畅自然。

### 参考品牌分析

| 品牌 | 借鉴元素 | 应用位置 |
|------|---------|---------|
| **Apple** | 极简主义、大量留白、精致细节 | 整体布局、组件设计 |
| **Google** | 现代清爽、Material Design 阴影 | 卡片样式、按钮设计 |
| **OpenAI** | 聚焦对话、中心输入框、绿色品牌色 | 主页布局、输入框设计 |
| **Claude** | 温暖色调、友好交互、清晰层次 | 配色方案、交互设计 |
| **Manus** | 对话界面、消息气泡、流式输出 | 聊天界面、消息展示 |

## 核心改进

### 1. 主页重设计

#### Hero 区域

主页采用全屏 Hero 区域设计，营造沉浸式体验。顶部是 Logo 和徽章的组合：一个紫色渐变的闪电图标，配合 "基于 P.R.O.M.P.T. 框架" 的白色徽章，既突出品牌特色，又强调技术优势。

标题使用超大字号 "AI Team"（6xl/7xl），配合副标题 "你的智能协作伙伴"，简洁有力地传达产品定位。下方的描述文字解释了核心价值："利用大模型实现代码自我进化，集成 Hugging Face 资源，打造真正智能的 AI 团队协作系统。"

#### 主输入框

输入框是整个页面的视觉焦点。采用大尺寸设计（3 行高），带有渐变边框（紫色到粉色），悬停时边框光晕增强，吸引用户注意。占位符文字 "和 AI Team 聊天... 试试：帮我生成一个天气查询工具" 既友好又具有引导性。

输入框下方是一个浅灰色的工具栏，左侧显示 "支持自然语言输入" 提示，右侧是紫色渐变的 "开始对话" 按钮，带有发送图标。整个输入区域使用了白色背景和微妙的阴影，营造出卡片悬浮的效果。

#### 快速操作

输入框下方是四个快速操作按钮：工具生成器、技能中心、自我进化。这些按钮采用白色背景、灰色边框，悬停时边框变为紫色，阴影加深，提供清晰的交互反馈。

#### 功能卡片区

向下滚动后，用户可以看到 "核心功能" 区域。四张功能卡片采用网格布局（响应式），每张卡片包含：
- 渐变图标（蓝色、紫色、绿色、橙色）
- 功能标题（智能对话、工具生成器、技能中心、自我进化）
- 简短描述
- "了解更多" 链接（带箭头图标）

卡片悬停时会轻微上浮（-4px），阴影加深，图标放大，箭头向右移动，提供丰富的视觉反馈。

#### 底部区域

页面底部是简洁的 Footer，包含 GitHub 链接、文档链接和版权信息。采用浅灰色背景和边框，与主体内容分离。

### 2. 聊天界面重设计

#### 布局结构

聊天界面采用经典的三栏布局：左侧侧边栏（可折叠）、中间主对话区、右侧预留扩展空间。

**侧边栏**（宽度 320px）：
- 顶部：紫色渐变的 "新建对话" 按钮
- 中间：历史对话列表，每个对话显示图标、标题、最后一条消息和时间戳
- 底部：版本信息 "AI Team v1.0"

**主对话区**：
- 顶部导航栏（高度 64px）：折叠按钮、助手信息（头像 + 名称 + 在线状态）、更多按钮
- 消息区域：最大宽度 768px，居中显示，背景为浅灰色
- 底部输入区：固定在底部，包含多行文本框和工具栏

#### 消息样式

**AI 消息**：
- 左对齐布局
- 紫色渐变圆角头像（10x10，圆角 12px）
- 白色气泡背景，灰色边框
- 左上角圆角较小（4px），营造对话气泡效果
- 文字颜色为深灰黑色

**用户消息**：
- 右对齐布局
- 灰色渐变圆角头像
- 紫色渐变气泡背景
- 白色文字
- 右上角圆角较小（4px）

**时间戳**：
- 小字号（12px）
- 浅灰色
- 根据消息角色左对齐或右对齐

#### 输入区域

输入区域采用现代化设计：
- 浅灰色背景的多行文本框，自动扩展高度（最小 56px，最大 200px）
- 左侧工具栏：附件按钮、表情按钮
- 右侧：紫色渐变的 "发送" 按钮，带图标
- 聚焦时边框变为紫色，带外发光效果
- 底部提示："AI Team 可能会出错。请核查重要信息。"

#### 交互细节

**侧边栏折叠**：点击菜单按钮，侧边栏宽度从 320px 过渡到 0px，持续时间 300ms，使用 ease 缓动函数。

**打字指示器**：AI 回复时显示三个跳动的小圆点，使用 CSS 动画，每个圆点延迟 0.2s 开始动画，营造打字效果。

**消息发送动画**：新消息从底部滑入，透明度从 0 到 1，持续时间 400ms。

### 3. 配色方案

#### 主色调

**品牌色**：`#6366F1`（紫蓝色）到 `#8B5CF6`（紫色）的渐变，用于主按钮、头像、强调元素。这个颜色既有科技感，又温暖友好。

**背景色系**：
- 主背景：`#FFFFFF`（纯白）
- 次级背景：`#F8F9FA`（极浅灰）
- 卡片背景：`#FFFFFF` with `box-shadow`

**文字色系**：
- 主文字：`#111827`（深灰黑）
- 次级文字：`#6B7280`（中灰）
- 占位符：`#9CA3AF`（浅灰）

**功能色**：
- 成功/在线：`#10B981`（绿色）
- 警告：`#F59E0B`（橙色）
- 错误：`#EF4444`（红色）

#### 渐变效果

**Hero 区域背景**：
```css
background: linear-gradient(135deg, #EEF2FF 0%, #FAF5FF 50%, #FDF2F8 100%);
```
从淡蓝色到淡紫色再到淡粉色，营造柔和的科技感。

**主按钮渐变**：
```css
background: linear-gradient(135deg, #6366F1 0%, #8B5CF6 100%);
```
从紫蓝色到紫色，视觉冲击力强。

**输入框边框渐变**（悬停时）：
```css
border-image: linear-gradient(135deg, #6366F1, #8B5CF6, #EC4899) 1;
```
从紫蓝色到紫色再到粉色，增加活力。

### 4. 组件设计

#### 按钮

**主按钮**：
- 背景：紫色渐变
- 圆角：12px
- 内边距：12px 24px
- 字体：14px 加粗
- 悬停：轻微放大（scale 1.05）+ 阴影加深

**次级按钮**：
- 背景：白色
- 边框：1px solid #E5E7EB
- 悬停：边框变为紫色 + 阴影

**图标按钮**：
- 圆形或圆角矩形
- 悬停：背景色变化

#### 卡片

**功能卡片**：
- 背景：白色
- 圆角：16px
- 阴影：`0 4px 6px rgba(0, 0, 0, 0.07)`
- 悬停：`0 10px 15px rgba(0, 0, 0, 0.1)` + 上移 4px
- 内边距：32px

**消息卡片**：
- 用户消息：紫色渐变背景，白色文字，右对齐
- AI 消息：白色背景，深色文字，左对齐
- 圆角：16px（外侧），4px（内侧尖角）
- 最大宽度：70%

#### 输入框

**单行输入**：
- 边框：1px solid #E5E7EB
- 圆角：12px
- 内边距：12px 16px
- 聚焦：边框变为紫色，带外发光

**多行输入**：
- 最小高度：56px
- 最大高度：200px
- 自动扩展

### 6. 响应式设计

#### 断点

- **移动端**：`< 640px`
- **平板**：`640px - 1024px`
- **桌面**：`> 1024px`

#### 适配策略

**移动端**：
- 单栏布局
- 侧边栏改为底部抽屉
- 输入框占满宽度
- 功能卡片堆叠显示
- Hero 区域标题缩小为 4xl

**平板**：
- 两栏布局
- 侧边栏可折叠
- 对话区域适当缩小
- 功能卡片 2 列显示

**桌面**：
- 三栏布局（可选）
- 侧边栏固定显示
- 对话区域最大宽度 800px
- 功能卡片 4 列显示

## 技术实现

### 技术栈

- **React 18**：UI 框架，使用函数组件和 Hooks
- **TypeScript**：类型安全，提高代码质量
- **Tailwind CSS**：样式框架，快速构建响应式界面
- **Lucide Icons**：图标库，提供一致的图标风格
- **Wouter**：轻量级路由库
- **Sonner**：Toast 通知库

### 关键组件

1. **Home.tsx**（主页）：
   - Hero 区域
   - 主输入框
   - 快速操作按钮
   - 功能卡片区
   - Footer

2. **Chat.tsx**（聊天界面）：
   - 侧边栏（历史对话列表）
   - 主对话区（消息列表）
   - 输入区域（多行文本框 + 工具栏）
   - 打字指示器

3. **App.tsx**（路由配置）：
   - 添加 `/chat` 路由
   - 将新主页设为默认路由

### 状态管理

- **React Context**：全局状态（用户信息、主题）
- **useState/useReducer**：组件状态
- **自定义 Hooks**：复用逻辑（useChat, useWebSocket）

### 代码优化

- 使用 TypeScript 接口定义数据结构
- 组件拆分，提高可维护性
- 使用 Tailwind 的 JIT 模式，减小 CSS 体积
- 懒加载路由组件
